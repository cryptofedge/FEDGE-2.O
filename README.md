## Quick Start

### Prerequisites

- Node.js 18+
- Python 3.10+
- OpenClaw runtime
- Anthropic API key
- WSL2 Ubuntu (recommended)

### Install

```bash
git clone https://github.com/cryptofedge/FEDGE-2.O.git
cd FEDGE-2.O
npm install
```

### Environment Setup

```bash
cp .env .env.local
# Add your Anthropic API key and any other credentials
```

### Run

```bash
node index.js
```
