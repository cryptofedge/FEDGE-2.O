# Release Playbook

## Standard Single Rollout (4-week timeline)

### Week 1 — Tease
- Day 1: Announce on IG stories (silhouette/cryptic visual)
- Day 3: 15-sec audio snippet on TikTok + IG reels
- Day 5: Submit to DistroKid (needs min. 7 days before release date)
- Day 7: Behind-the-scenes content (studio session, recording clip)

### Week 2 — Build
- Day 8: Cover art reveal on all platforms
- Day 10: Pitch to Spotify editorial playlists (via Spotify for Artists — submit 7+ days before release)
- Day 11: Lyrics/title teaser on Twitter/X
- Day 14: Pre-save link live (DistroKid HyperFollow)

### Week 3 — Release Week
- Day 15 (Tuesday): Press outreach to blogs/media
- Day 17 (Thursday): Reminder posts, countdown stories
- Day 18 (Friday): RELEASE DAY
  - Post everywhere simultaneously
  - Go Live on IG
  - Pin post on all platforms
  - Send WhatsApp broadcast to fanbase
  - Thank collaborators publicly, tag them

### Week 4 — Push
- Day 21: Share fan reactions/reposts
- Day 23: TikTok challenge or use-my-sound push
- Day 25: Pitch to independent playlist curators
- Day 28: Performance clip or lyric video drop

## Release Day Checklist
- [ ] All platforms live (Spotify, Apple Music, YouTube Music, Amazon)
- [ ] YouTube music video or visualizer uploaded
- [ ] IG feed post + story + reels
- [ ] TikTok post
- [ ] Twitter/X announcement
- [ ] Facebook post
- [ ] WhatsApp status
- [ ] Link in bio updated
- [ ] Pre-save flipped to stream link
