# HyperFrames — Edit Videos by Vibe-Coding

HyperFrames lets AI agents compose videos by writing HTML, CSS & JS.
The bridge between code and video production.
URL: https://hyperframes.heygen.com
GitHub: https://github.com/heygen-com/hyperframes (Open Source — Apache 2.0)

---

## What Makes HyperFrames Unique

Traditional video editing: drag-and-drop, timeline-based, manual
HyperFrames: *describe the video in code → AI composes it*

This means FEDGE 2.O and Claude Code can compose and edit videos
programmatically — no human editor needed.

---

## How It Works

1. Write a composition in HTML/CSS/JS (like building a webpage)
2. HyperFrames renders it as a video
3. AI agents (Claude Code, FEDGE 2.O) can write and edit these compositions
4. Output: finished video ready to publish

---

## Key Concept: Compositions

A composition is an HTML file that describes your video:
- Layout, timing, transitions
- Text overlays, graphics, animations
- Product placements, branding elements
- Multiple scenes chained together

```html
<!-- Example composition: product-promo.html -->
<scene duration="3s">
  <product src="eyeshadow.jpg" position="center" />
  <text>New Drop: Midnight Palette</text>
  <music src="brand-track.mp3" />
</scene>
```

## Getting Started

1. Install HyperFrames:
```bash
npx hyperframes install
```

2. Start with a template:
```bash
npx hyperframes create product-promo
```

3. Edit composition HTML → preview → export video

4. Use with Claude Code for AI-assisted editing
