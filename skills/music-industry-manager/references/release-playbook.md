# Release Playbook — Full Rollout Template

## Timeline (6 Weeks Out)

### Week 6 (Pre-Launch)
- [ ] Finalize master recording and artwork
- [ ] Submit to distributor (DistroKid, TuneCore, etc.) — set release date
- [ ] Register song with PRO (ASCAP/BMI) and publisher
- [ ] Submit to Spotify for Artists playlist pitching (must be 7+ days before release)
- [ ] Create pre-save link (Feature.fm, Linkfire, Toneden)
- [ ] Write press release draft

### Week 5
- [ ] Launch pre-save campaign on all socials
- [ ] Start teaser content: behind-the-scenes, snippet reels
- [ ] Pitch to music blogs and playlist curators
- [ ] Send press release to media contacts
- [ ] Set up YouTube premiere or lyric video

### Week 4
- [ ] Ramp up content: 1 post/day minimum
- [ ] Run paid pre-save ads (Meta + TikTok)
- [ ] Confirm music video release plan (same day or +1 week)
- [ ] Outreach to influencers for UGC (user-generated content)

### Week 3
- [ ] Countdown content begins
- [ ] Podcast/interview outreach
- [ ] Finalize merch drops tied to release (if applicable)

### Week 2
- [ ] "Last week before drop" content push
- [ ] Repost fan reactions to pre-save snippets
- [ ] Final ad creative refresh

### Release Week
- [ ] Drop day: post across ALL platforms simultaneously
- [ ] Go Live on Instagram/TikTok day of release
- [ ] Ask fans to add to library, share, playlist
- [ ] Monitor Spotify for Artists dashboard hourly first 48h
- [ ] Pitch to editorial playlists post-release via curators

### Post-Release (Weeks 1–4)
- [ ] Weekly streaming report review
- [ ] Re-pitch to playlists with updated stream stats
- [ ] Create reaction content, fan features
- [ ] Remix / acoustic version consideration
- [ ] Sync licensing pitches begin
