# Claude Design — AI Design Tool by Anthropic Labs

Claude Design lets you create designs, interactive prototypes, presentations, and more
by having a conversation with Claude.
URL: https://claude.ai (Anthropic Labs experimental preview)

## How It Works

Two main areas:
- *Chat interface (left)* — describe what you want
- *Canvas (right)* — Claude generates a working design

Describe → Generate → Iterate through conversation and inline comments.

## Key Features

### 🎨 What You Can Create
- UI/UX designs and wireframes
- Interactive prototypes
- Presentations and slide decks
- Landing pages
- App mockups
- Marketing visuals

### 🏢 Design System Integration
- Automatically inherits your org's brand colors, fonts, and components
- No need to upload brand assets manually
- Everything stays on-brand from the start

### 💬 Two Ways to Iterate
- *Chat* — broad changes (layout shifts, new sections, aesthetic changes)
- *Inline comments* — click directly on canvas elements for targeted edits

### 📤 Export & Share
- Export in multiple formats (PDF, image, etc.)
- Share via link: view-only, comment, or edit access
- Present directly from Claude Design

### 🔄 Version Management
- Save multiple directions without losing work
- Tell Claude: "Save this and try a completely different approach"

## Writing Effective Prompts

A great prompt includes:
1. *Goal* — what you're building
2. *Layout* — how things should be arranged
3. *Content* — what information to display
4. *Audience* — who will use it

Example:
"Design a product landing page for Sensual Make Up's new eyeshadow palette.
Bold, feminine aesthetic. Above fold: hero image, headline, CTA button.
Below: 3 feature callouts, reviews section, buy now."
