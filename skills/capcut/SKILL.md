## Content Workflows

### Music Video Promo Reel (Mando)
1. Import performance clip or music video snippet
2. Set audio to "MOMENTOS" track
3. Auto-sync cuts to beats
4. Add auto-captions for lyrics
5. Color grade (cinematic look)
6. Add artist name text overlay
7. Export 9:16 for TikTok/Reels

### Product Tutorial (Sensual Make Up)
1. Film product application walkthrough
2. Speed up boring parts (2x-3x)
3. Add auto-captions
4. Overlay product name + price text
5. Add upbeat background music
6. Export square (1:1) for feed OR 9:16 for Reels/TikTok

### Event Recap
1. Compile show footage clips
2. Beat sync to song
3. Add location/date text overlays
4. Fast-paced edit (2-3 seconds per clip)
5. End card with streaming links
