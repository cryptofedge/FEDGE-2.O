# Promo Templates

## Instagram Caption (Single Release)
```
🔥 [SONG TITLE] — OUT NOW 🔥

[1-2 lines describing the vibe/story of the song]

Link in bio 👆 | Stream on all platforms

🎵 [ARTIST] feat. [COLLAB if applicable]
#MandoElPelado #LatinUrban #NuevoTema #Ecuador #NYC #Reggaeton
```

## TikTok Hook Script (15 sec)
```
[0-3s]: Bold visual or movement
[3-8s]: Drop the best 8 bars of the track
[8-13s]: Text overlay: "NEW SONG OUT NOW 🔥"
[13-15s]: CTA: "Link in bio / Search [song name]"
```

## Instagram Story Sequence (Release Day)
1. Countdown sticker → "TODAY'S THE DAY"
2. Cover art + swipe up/link sticker
3. 15-sec clip of the track
4. Poll: "You vibing? 🔥 or 🔥🔥"
5. Repost fan reactions
