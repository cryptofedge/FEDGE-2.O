# Claude Code — Agentic Coding Tool

Claude Code reads your codebase, edits files, runs commands, and integrates with your
development tools. Available in terminal, IDE, desktop app, and browser.
URL: https://code.claude.com

## Installation

### Terminal (macOS/Linux)
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

### Windows (PowerShell)
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Homebrew
```bash
brew install --cask claude-code
```

### WinGet
```bash
winget install Anthropic.ClaudeCode
```

### IDE Extensions
- *VS Code* — Install from Marketplace: `anthropic.claude-code`
- *JetBrains* — Available for IntelliJ, PyCharm, WebStorm, etc.

---

## What Claude Code Can Do

- Read and understand entire codebases
- Edit files across multiple directories
- Run shell commands and scripts
- Write tests and fix bugs
- Refactor and restructure code
- Set up new projects from scratch
- Integrate with Git, npm, pip, etc.
- Debug errors with full context
