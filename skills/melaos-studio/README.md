# Melao's Studio

> *"Every idea deserves a beat. Every feeling deserves a song."*

A FEDGE 2.O skill that turns WhatsApp messages into real songs via Suno AI.

## Full Repo
https://github.com/cryptofedge/melaos-studio

## Quick Setup
1. Clone melaos-studio repo
2. Copy `skills/melaos-studio.skill.js` → `FEDGE-2.O/agent/skills/`
3. Add `MEMORY_ENCRYPTION_KEY` to `.env`
4. Users link Suno: `LINK email password`
