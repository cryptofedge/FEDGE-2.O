# Claude Cowork — Agentic Task Execution

Claude Cowork brings Claude Code's agentic capabilities to Claude Desktop for knowledge
work beyond coding. Describe a task, step away, come back to finished work.
URL: https://claude.com/product/cowork

## What Makes Cowork Different

Regular Claude: one prompt → one response
Cowork: one goal → full multi-step execution with file access and real changes

You describe an outcome. Claude plans, executes, and delivers — formatted documents,
organized files, synthesized research, and more.

---

## Key Capabilities

### 📁 File & Document Management
- Organize and rename files across folders
- Convert between formats
- Batch process documents
- Clean up and restructure directories

### 🔬 Research & Analysis
- Search, synthesize, and summarize information
- Analyze data from files
- Generate reports with findings

### 📝 Document Creation
- Draft formatted documents, reports, proposals
- Build presentations from raw notes
- Create templates from scratch

### ⏰ Scheduled Tasks
- Set up recurring automated tasks
- Type `/schedule` in any Cowork task
- Runs automatically while your computer is awake

### 🔌 Plugins
- Install plugins that bundle skills + connectors + sub-agents
- Customize Claude for your role, team, or company

## How to Use

1. Open Claude Desktop
2. Start a Cowork session
3. Describe your task/outcome
4. Set approval mode: *supervised* (Claude asks) or *autonomous* (Claude runs)
5. Step away or monitor progress
6. Return to finished work

---

## Permission Modes
- *Supervised* — Claude asks for approval before taking action
- *Autonomous* — Claude runs independently, only asks before permanent deletes

## Availability
- Paid plans only: Pro, Max, Team, Enterprise

## Important Notes
- Computer must stay awake for scheduled tasks to run
- Claude can make real changes to files you share
- Review what folders/files you grant access to
- Use standard chat for simple tasks; save Cowork for complex multi-step work
